# Acta de Constitución

Este documento registra todos los elementos necesarios para dar conformidad al inicio del proyecto, explicando datos del proyecto, responsables y alcances.

| Resumen              |                                                                                |
| -------------------- | ------------------------------------------------------------------------------ |
| Proyecto             | Sistema de Gestión para Mercados Municipales.                                  |
| Cliente              | Alcaldía de Sucre                                                              |
| Objetivo             | Digitalizar la administración de los mercados Campesino, Central y Entre Ríos. |
| Presupuesto          | 70000 Bs.                                                                      |
| Duración             | 6 meses                                                                        |
| Entregable principal | MVP funcional                                                                  |

## Objetivo

Digitalizar la administración de los mercados municipales, realizando el control de los puestos y comerciantes autorizados dentro de estos mismos, controlando tarifas, licencias sanitarias y proporcionando informes mensuales.

## Alcance

El proyecto INCLUYE:

- Registro de puestos y comerciantes autorizados.
- Registro de cobro de tarifas.
- Control de licencias sanitarias.
- Generación de informes mensuales para la dirección de ingresos.
- Acceso para administrativos y encargados de los controles anteriores.

El proyecto NO INCLUYE:

- Acceso para comerciantes.
- Geolocalización.

El proyecto está destinado a ejecutarse en **6 meses**.

## Actores Involucrados e Interesados

- Alcaldía de Sucre
- Comerciantes
- Encargado del Proyecto
- Equipo de Desarrollo
