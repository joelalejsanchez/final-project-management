# Archivos

Los puntos especificados en el trabajo se organizan dentro de 8 documentos.

1. Acta de constitución
2. Descripción del Proyecto
3. Solicitud de Proyecto
4. Documento Inception
5. Plan de mejora continua
6. Solicitud de cambios
7. Acta de cierre
8. Cierre de proyecto
9. Informe de avance

Los puntos se distribuyen exactamente como en la siguiente tabla:
