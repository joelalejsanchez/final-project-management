# Archivos

Los puntos especificados en el documento se distribuyen entre 6 archivos:

1. Acta de Constitución
   - Resumen del proyecto (Proyecto, Cliente, Objetivo, Presupuesto, Duración, Entregable)
   - Objetivo
   - Alcance
   - Actores Involucrados e interesados
2. Descripción del Proyecto
   - Metodología
   - Requerimientos
   - Matriz de priorización
   - Cronograma
   - Análisis de riesgos
   - Plan de Recursos
   - Presupuesto
3. Solicitud de Proyecto
4. Documento de Inception
5. Informe de Avance
   - Avances
   - Tablero de seguimiento
   - Registro de incidencias
6. Solicitud de Cambios
   - Proceso de negociación
   - Actualización de Riesgos
7. Cierre de proyecto
   - Acta de Cierre
   - Validación del Producto
   - Lecciones Aprendidas
   - Evaluación 4P
   - Value Stream Mapping
   - Plan de Escalado Organizacional
